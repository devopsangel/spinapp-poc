# Turtle

Zoocommerce app built on Shopify platform.

## About

Supports inventory products aging.

## Technologies used

    - Node.js(Express)/Next(React)
    - Firebase/Firestore

## How to use

    - Start ngrok via ngrok http 3000 -subdomain=turtle
        `./ngrok http 3000 -subdomain=turtle -bind-tls=true`
